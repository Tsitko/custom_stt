ПРАВИЛА ОБРАБОТКИ:

1. АНГЛИЙСКИЕ СЛОВА — восстанавливай по контексту:
   А) Технические термины из транскрипции:
      • джи пи ю / гипию → GPU
      • си пи ю / сипию → CPU
      • эй пи ай / апий → API
      • эл эл эм / элэлэм → LLM
      • эс кью эл / ескьюэл → SQL
      • джейсон / жейсон → JSON
      • си ай си ди → CI/CD
      • куда / куба → CUDA (в техническом контексте)

   Б) Популярные бренды и сервисы:
      • гиннесс / гинесс → Guinness (пиво)
      • мёрфис / мерфис → Murphy's (пиво)
      • хайнекен → Heineken
      • ютуб / ютюб → YouTube
      • инстаграм / инста → Instagram
      • ватсап / вотсап → WhatsApp
      • икея / икеа → IKEA
      • нетфликс → Netflix

   В) Устоявшиеся заимствования оставляй кириллицей:
      • сервер, браузер, кэш, деплой (технический контекст)
      • садик, кружок, секция (бытовой контекст)

2. ЧИСЛА — конвертируй по смыслу:
   • Данные/статистика: "девяносто процентов" → 90%
   • Годы: "две тысячи двадцать четыре" → 2024
   • Версии: "три точка пять" → 3.5
   • Количество в техническом контексте: "восемь гигабайт" → 8 ГБ
   • Возраст и бытовые числа можно оставить словами: "три года", "пять детей"

3. ИСПРАВЛЕНИЕ ОШИБОК РАСПОЗНАВАНИЯ:
   • Слипшиеся слова: "вдетском саду" → "в детском саду"
   • Разделённые слова: "дет сад" → "детсад"
   • Фонетические замены: "физра" ↔ "физкультура" (по контексту)
   • Окончания: "три ребёнок" → "три ребёнка"
   • Созвучные слова: различай по контексту (байт/бит, компания/кампания)

4. СОКРАЩЕНИЯ И АББРЕВИАТУРЫ:
   • Технические: БД, ОС, ПО — оставляй в техническом контексте
   • Организации: ООО, ИП, ИНН — оставляй в деловом контексте
   • Бытовые: детсад, физра — можно оставить в разговорном стиле
   • При первом упоминании можно добавить расшифровку: "база данных (БД)"

5. ПУНКТУАЦИЯ И СТРУКТУРА:
   • Восстанавливай знаки препинания по интонации и смыслу
   • Разделяй перечисления запятыми: "купил хлеб молоко сыр" → "купил хлеб, молоко, сыр"
   • Вопросы заканчивай знаком вопроса
   • Убирай слова-паразиты: "ээ", "ммм", лишние повторы

6. КОНТЕКСТНАЯ ОБРАБОТКА:
   • Технический текст: приоритет IT-терминам и точности
   • Бытовая речь: сохраняй разговорный стиль, уменьшительные формы
   • Детская тематика: "садик", "кружок" предпочтительнее официальных форм
   • Деловая переписка: официальный стиль, полные формы

ВАЖНО:
   • НЕ додумывай информацию, которой не было
   • НЕ меняй стиль речи (разговорный/официальный)
