Нарезать любой входной текст (документ, лог, отчет) в чистый, однозначный вариант, готовый к синтезу речи. Выполнить:

1. Развёртывание сокращений  
   • «НДС» → «налог на добавленную стоимость»  
   • «УСН» → «упрощённая система налогообложения»  
   • «ИП» → «индивидуальный предприниматель»  
   • «ГЧП» → «государственно‑частичное партнёрство»  
   • «ПФ» → «Пенсионный фонд»  

2. Транслитерация английских терминов  
   • «GDP» → «джи ди пи», «VAT» → «ват», «IRS» → «ай ар эс», «API» → «а пи», «CI» → «си ай», «LLM» → «эл эл эм», «CPU» → «си пи ю».  

3. Письменность чисел  
   • Все цифры писать словами: «три» вместо «3», «двенадцать» вместо «12», «две тысячи двадцать три» вместо «2023», «четыре целых пятьдесят шесть сотых» вместо «4.56».  

4. Пунктуация и синтаксис  
   • Запятые ставить в списках и после вводных конструкций.  
   • Разбивать длинные предложения точками с запятой, если в них несколько действий.  
   • Добавлять уточняющие слова для однозначности (например, «срок сдачи» → «срок сдачи декларации»).  

5. Орфография  
   • Исправлять типичные ошибки, связанные с фонетическим распознаванием (напр., «налоговый» вместо «налоговий»).  

Формат ответа  
Только окончательный текст, без комментариев, примеров, инструкций, таблиц или код‑блоков.
